# 2026 寰辰早會流程核對系統

本專案是為 2026 寰辰早會的 **音控人員** 與 **主持人員** 設計的早會流程核對系統。提供同仁在早會前與早會進行中核對各項細節，確保流程順暢；並提供後台供管理人員查看確認狀態。

---

## 🌟 系統特色

1. **精美現代化介面**：採用深色玻璃擬物化 (Glassmorphic) 風格，具備動態漸層與平滑微動畫，支援手機與電腦版響應式排版。
2. **動態進度追蹤**：上方設有發光進度條，實時計算核對完成度，只有當 100% 完成所有可見項目並勾選「確認清楚了解流程」後，才能提交。
3. **智慧星期篩選**：自動偵測當天是星期幾，預設顯示當天日期的流程，簡化核對內容；也可隨時切換查看其他天或顯示全部。
4. **完整管理後台**：
   - 儀表板統計今日提交總數、音控提交數、主持提交數。
   - 支援即時姓名關鍵字搜尋與角色篩選。
   - 點擊「檢視細節」可查看每位同仁核對時**具體勾選了哪些項目**（以及跳過了哪些項目）。
   - 提供「刪除」功能方便清理測試數據。
5. **本地 JSON 資料庫**：無需複雜資料庫安裝，數據直接保存在專案目錄下的 `db.json` 中，輕量且防丟。

## 🚀 如何運行與使用

### 1. 安裝與啟動
在專案根目錄下，您可以使用以下指令來啟動伺服器：

```bash
# 啟動伺服器
npm start
```

如果是在開發期間，想要程式碼變更時自動重啟，可以使用：
```bash
npm run dev
```

### 2. 訪問系統網址
打開瀏覽器，訪問以下網址：
- **同仁核對首頁**：[http://localhost:3000](http://localhost:3000)
- **管理員後台**：[http://localhost:3000/admin.html](http://localhost:3000/admin.html)

## 🛠️ 技術說明

- **前端**：純 HTML5 + CSS3 (Vanilla CSS) + JavaScript，無第三方打包框架，載入極速。圖示使用 FontAwesome。
- **後端**：Node.js + Express，提供 `/api/submit` (提交)、`/api/submissions` (查詢) 與 `/api/delete` (刪除) API。
- **持久化**：使用 node 的原生 `fs` 模組讀寫 `db.json`，系統會自動在初次啟動時初始化該檔案。
